# クイックスタートガイド

このガイドでは、MBTI性格診断アプリケーションを最短で起動する方法を説明します。

## 📋 前提条件

以下がインストールされていることを確認してください：

- **Docker Desktop**: [公式サイト](https://www.docker.com/products/docker-desktop)からダウンロード
- **Git**: バージョン管理システム

## 🚀 5分で起動

### ステップ1: リポジトリのクローン

```bash
git clone <repository-url>
cd traning-mbti
```

### ステップ2: 起動

```bash
./start.sh
```

このコマンド一つで以下が自動実行されます：

✅ 環境設定ファイル（.env）の作成  
✅ Dockerコンテナのビルド  
✅ データベースの初期化  
✅ アプリケーションの起動  
✅ ヘルスチェック  

### ステップ3: アクセス

起動が完了したら、ブラウザで以下のURLにアクセスしてください：

**メインページ**: http://localhost:5000

## 🎯 基本的な使い方

### 診断を受ける

1. トップページで「診断を始める」ボタンをクリック
2. 12問の質問に順番に回答
3. 診断結果が表示されます

### 診断履歴を見る

http://localhost:5000/admin/history にアクセス

## 🛠️ よく使うコマンド

```bash
# ログをリアルタイムで確認
./logs.sh

# アプリケーションを停止
./stop.sh

# 完全にリセット（データも削除）
./stop.sh
# プロンプトで 'yes' と入力
```

## 🐛 トラブルシューティング

### ポート5000が既に使用されている

```bash
# 使用中のプロセスを確認
lsof -i :5000

# プロセスを終了してから再度起動
./start.sh
```

### データベースの接続エラー

```bash
# コンテナを完全に削除して再起動
docker-compose down -v
./start.sh
```

### Dockerが起動しない

Docker Desktopが起動していることを確認してください。

## 💡 ヒント

- **キーボードショートカット**: 質問ページで「1」「2」キーを押すと選択肢を選べます
- **レスポンシブ対応**: スマートフォンからもアクセスできます
- **診断履歴**: 全ての診断結果はデータベースに保存されます

## ❓ よくある質問

**Q: データはどこに保存されますか？**  
A: Dockerボリュームに保存されます。`docker-compose down -v` で削除されます。

**Q: 本番環境で使えますか？**  
A: いいえ、これは教育目的のデモです。セキュリティ脆弱性が意図的に含まれています。

**Q: カスタマイズできますか？**  
A: はい！質問内容は `app/quiz_data.py` で変更できます。
