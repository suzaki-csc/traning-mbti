# 効果音ファイルについて

このディレクトリには、クイズアプリで使用する効果音ファイルを配置します。

## 必要なファイル

- `correct.mp3` - 正解時の効果音
- `incorrect.mp3` - 不正解時の効果音

## 効果音の取得方法

1. フリーの効果音サイトからダウンロード
   - 例: Freesound.org, 効果音ラボ など
2. 適切なライセンス（商用利用可、または教育目的利用可）のものを選択
3. MP3形式で保存
4. ファイル名を `correct.mp3` と `incorrect.mp3` にリネーム

## 注意事項

- 効果音ファイルがなくてもアプリケーションは正常に動作します（音が鳴らないだけです）
- 効果音ファイルは `.gitignore` に含まれているため、Gitにはコミットされません
- 本番環境では、適切な効果音ファイルを配置してください
